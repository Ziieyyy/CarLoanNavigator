# Deploying to Wasmer.io

This guide will help you deploy your Car Loan Navigator application to Wasmer.io.

## Prerequisites

1. Install Wasmer CLI:
   ```bash
   curl https://get.wasmer.io -sSfL | sh
   ```

   Or on Windows, you can use the provided PowerShell script:
   ```powershell
   .\install-wasmer.ps1
   ```

   Alternatively, download from: https://github.com/wasmerio/wasmer/releases

2. Create an account at https://wasmer.io

## Deployment Steps

1. **Build the application** (if you haven't already):
   ```bash
   npm run build
   ```

2. **Initialize Wasmer package** (if you haven't already):
   ```bash
   wasmer init
   ```

3. **Login to Wasmer**:
   ```bash
   wasmer login
   ```

4. **Deploy to Wasmer**:
   ```bash
   wasmer publish
   ```

## Configuration

The deployment is configured through:
- `wasmer.toml`: Package configuration
- `dist/`: Built static files (created by `npm run build`)

## Custom Domain (Optional)

After deployment, you can add a custom domain through the Wasmer dashboard:
1. Go to https://wasmer.io/dashboard
2. Select your package
3. Navigate to the "Domains" section
4. Add your custom domain

## Updating the Deployment

To update your deployment after making changes:
1. Rebuild the application: `npm run build`
2. Redeploy: `wasmer publish`

## Troubleshooting

If you encounter issues:
1. Ensure all files are committed to Git
2. Check that `dist/` folder is generated by the build process
3. Verify `wasmer.toml` configuration is correct
4. Make sure you're logged in to Wasmer: `wasmer login`
